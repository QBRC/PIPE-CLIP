PIPE-CLIP
=========

Pipeline for CLIP-seq Analysis.
Galaxy site: http://pipeclip.qbrc.org/

Requirement:
(1) Python 3.0 and above; 
(2) R 3.0 and above;
(3) Python packages: pysam. pybedtools, matplotlib, numpy, ghmm and rpy2;
(4) R packages: VGAM and its dependencies.
